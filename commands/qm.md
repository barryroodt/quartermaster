# /qm

Dispatches to the quartermaster Bun CLI. Subcommands: `init`, `survey <goal>`, `list [--source-type=X]`, `trust add|list`, `prune`.

Run:

```
bun ${CLAUDE_PLUGIN_ROOT}/src/cli.ts $ARGUMENTS
```
